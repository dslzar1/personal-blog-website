# Personal Blog Website

A multi-page blog-style website built using HTML, CSS, and JavaScript.

## Features
- Home, About, Blogs, and Movies pages
- Embedded media content
- Basic interactive UI elements using JavaScript
- Custom styling with CSS

## How to Run
Open `index.html` in a web browser.
